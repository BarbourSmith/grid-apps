    Tool Options

    Tool options are only relevant for CAM mode. If you aren't using CAM mode, this section can be ignored.
    Each CAM Operation has a tool parameter, which is a refrence to a tool ID in the tool list (not to be confused with the tool number). the setTools function accepts an array of objects with the following parameters:
    Parameter Default Value
    id1000
    number1
    type"endmill"
    name"end 1/4"
    metricfalse
    shaft_diam0.25
    shaft_len1
    flute_diam0.25
    flute_len2
    taper_tip0
    order5

    Process Options

    The process options for a given mode determine the global parameters for how an object is processed. Some process options are mode-specific, and some are shared between modes.
